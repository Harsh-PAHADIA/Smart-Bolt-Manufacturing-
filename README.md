
# Smart Bolt Manufacturing System

A dashboard to monitor bolt manufacturing processes including:

* Production KPIs
* Defect analysis
* Workflow visualization
* Standard work documentation

## Project Overview

This project simulates a manufacturing monitoring system for bolt production.
It helps visualize the production workflow and key manufacturing metrics used in industrial environments.

The goal of this system is to demonstrate how software dashboards can assist engineers and manufacturing supervisors in monitoring production processes and improving efficiency.

## Manufacturing Workflow

The simulated manufacturing process includes the following stages:

Raw Material → Cold Forging → Thread Rolling → Heat Treatment → Quality Inspection

Each stage represents a critical step in transforming raw steel material into finished high-strength bolts.

## Features

### Production KPI Monitoring

Displays important manufacturing indicators such as:

* Production rate
* Machine utilization
* Defect rate

### Defect Analysis

Provides visualization tools for identifying and analyzing production defects.

### Workflow Visualization

Shows the step-by-step process of bolt manufacturing in a production environment.

### Standard Work Documentation

Provides structured documentation describing operational procedures used in manufacturing.
